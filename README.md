# 📊 Customer Behavior Analysis

## 📌 Project Overview

This project was completed as part of the **Alfido Tech Data Analytics Internship** through the **InternSpark** platform.

In this project, I analyzed customer purchasing behavior using Python to better understand customer patterns, identify different customer segments, and gain useful business insights through data visualization and machine learning techniques.

---

## 🎯 Objectives

- Clean and preprocess the customer dataset.
- Perform Exploratory Data Analysis (EDA).
- Segment customers using K-Means Clustering.
- Understand customer purchasing behavior through visualizations.
- Provide business recommendations based on the analysis.

---

## 📂 Dataset

**Source:** Kaggle

**Dataset Used:** `ecommerce_customer_data_large.csv`

The dataset contains customer purchase information such as age, gender, product category, purchase amount, payment method, and churn details.

---

## 📊 Project Workflow

The following steps were performed during the analysis:

- Data Cleaning and Preprocessing
- Missing Value Analysis
- Duplicate Record Check
- Feature Engineering
- Summary Statistics
- Customer Segmentation using K-Means Clustering
- Data Visualization
- Business Insights and Recommendations

---

## 📈 Visualizations

The notebook includes the following visualizations:

- Customer Age Distribution
- Product Category Distribution
- Gender-wise Purchase Analysis
- Monthly Purchase Trend
- Age Group Analysis
- Customer Segmentation using K-Means Clustering

---

## 🔍 Key Findings

- Customer purchasing behavior varies across different age groups.
- Some product categories are purchased more frequently than others.
- Purchase amounts differ among customer segments.
- K-Means clustering grouped customers with similar purchasing patterns, making it easier to understand different customer groups.
- The visualizations helped identify trends that can support better business decisions.

---

## 💡 Recommendations

- Personalize marketing campaigns for different customer segments.
- Promote frequently purchased product categories.
- Improve customer engagement through targeted recommendations.
- Monitor customer behavior regularly to identify changing trends.
- Use customer segmentation to support future business strategies.
